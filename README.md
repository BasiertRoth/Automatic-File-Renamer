# Automatic-File-Renamer

A simple Python script that renames all files in a folder based on a custom naming pattern.

I made this around 2022 for a GTA RP server, where clothing files needed to follow a specific naming format (e.g. a prefix or structured names).

Not heavily maintained, but might still be useful for simple tasks lol.
